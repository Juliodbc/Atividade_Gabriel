Comandos Básicos do Ubuntu

1. Atualizar o sistema

Para atualizar a lista de pacotes e instalar as atualizações disponíveis, use o seguinte comando:

sudo apt update && sudo apt upgrade

2. Instalar um pacote

Para instalar um novo pacote, utilize o comando:

sudo apt install nome-do-pacote

3. Remover um pacote

Para remover um pacote que não é mais necessário, utilize:

sudo apt remove nome-do-pacote

4. Verificar o espaço em disco

Para verificar o espaço disponível em disco, utilize:

df -h

5. Navegar entre diretórios

Para mudar de diretório, use o comando:

cd caminho/do/diretorio
